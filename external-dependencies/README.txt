This directory contains library code written outside of Intel. Developers are
encouraged to leverage useful and well tested libraries. Please Pay attention 
to licensing terms. Each case is different, but libraries that use LGPL or 
Boost licenses, or their equivalents, are preferred. 

What is in this directory:

There are a lot of files here, but only the following directories are currently
referenced by the unified build: 

/min_boost_1_46_1
This directory contains a sub set of the Boost distribution, specifically the
files used for boost::program_options
