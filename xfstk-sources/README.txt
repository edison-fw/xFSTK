This folder contains code written by Intel employees and contractors for
the xFSTK project (Cross Platform Firmware Software Tool Kit)

If the code is written by a third party and not owned by Intel 
(such as open source software) then it belongs in the external
dependencies folder located in a different directory.

What is in this directory:

/api	
This directory contains code that allows other developers to perform
operations using the xFSTK code base.

/core
This directory contains code that provides the major functionality of XFSTK.

/gui
This directory contains code used to create graphical user interfaces.

/installer
This directory contains code used to create platform specific 
installer packages.
